== Description ==
Gestisci e mostra la libreria di libri letti dal blogger con recupero delle informazioni del titolo dalle API Google Books e generazione dei link affiliato con API Amazon.
== Installation ==
1. Carica tutto in `/wp-content/plugins/book-library-plugin/`
2. Attiva il plugin
3. Imposta le API in **Impostazioni > Book Library**
